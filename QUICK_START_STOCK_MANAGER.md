# 🚀 Guide Rapide: Créer un Gestionnaire Stock (3 étapes)

## Pour l'Assistant Comptable (Gestionnaire/Manager)

Vous avez maintenant la possibilité de créer des utilisateurs qui gèrent **exclusivement** le Stock.

---

## 3 Étapes simples

### 📌 Étape 1: Aller à l'Administration
1. Cliquez sur **Administration** dans la barre de navigation
2. Vous êtes dans le **Dashboard admin**
3. Sélectionnez l'onglet **Utilisateurs**

### ➕ Étape 2: Remplir le formulaire
Dans la section **Créer un nouvel utilisateur**, entrez:

| Champ | Exemple |
|-------|---------|
| **Nom d'utilisateur** | `stock_user1` ou `magasin_dakar` |
| **Mot de passe** | `MyStock@2025` (min 6 caractères) |
| **Rôle** | 📦 **Gestionnaire Stock** |

### 💾 Étape 3: Créer
1. Cliquez le bouton **Créer**
2. Vous verrez le message: ✅ "Utilisateur créé avec succès"
3. L'utilisateur est créé et peut se connecter immédiatement

---

## Ce que cet utilisateur peut faire

### ✅ Autorisé
- 📋 **Consulter le Stock** - Voir la liste complète
- ➕ **Enregistrer** - Ajouter des opérations (entrée, sortie)
- ✏️ **Modifier** - Éditer des opérations existantes
- 🗑️ **Supprimer** - Retirer des opérations
- 📄 **Exporter** - Télécharger des rapports en PDF/HTML

### ❌ Interdit (Non accessible)
- Dashboard, Journal, Balance, Grand Livre, Relevé, Gestion d'utilisateurs

---

## Exemple de création réelle

```
Créer pour: Responsable du magasin de Dakar

1️⃣ Nom d'utilisateur: magasin_dakar
2️⃣ Mot de passe: Dakar2025#Stock
3️⃣ Rôle: 📦 Gestionnaire Stock
4️⃣ Clic → Créer ✓

Résultat: L'utilisateur magasin_dakar peut maintenant:
- Se connecter
- Accéder UNIQUEMENT au module Stock
- Gérer toutes les opérations de stock
```

---

## Modifier ou Supprimer

### Modifier
1. Onglet Utilisateurs
2. Cliquez **Modifier** sur l'utilisateur
3. Changez le rôle ou réinitialisez le mot de passe
4. Enregistrez

### Supprimer
1. Onglet Utilisateurs
2. Cliquez l'icône **Corbeille** 🗑️
3. Confirmez

---

## Support rapide

| Problème | Solution |
|----------|----------|
| "Nom d'utilisateur invalide" | Utilisez 3-32 caractères, lettres/chiffres/_ uniquement |
| "Mot de passe trop court" | Minimum 6 caractères |
| "Je ne vois pas Gestionnaire Stock" | Vérifiez que la page se charge correctement (F5) |
| "L'utilisateur ne peut pas accéder au Stock" | Assurez-vous que le rôle est bien "Gestionnaire Stock" |
